# Escapegineers

A Purdue-inspired escape room website built with Next.js and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS with Purdue-inspired theme
- **Language**: TypeScript
- **Fonts**: Cinzel (headings) and Inter (body)

## Purdue Theme Colors

- **Boiler**: #121212 (Dark background)
- **Gold Plus**: #C29A3A (Old Gold++)
- **Foundry**: #4F4F51 (Soft gray)
- **Ivory**: #F4F1E6 (Light paper)
- **Magenta**: #CA2E55 (Vibrant pink-red)
